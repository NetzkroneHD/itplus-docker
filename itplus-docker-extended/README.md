## Docker-Compose Cheatsheet

### Compose starten:  
`docker compose up [OPTIONS]`

Beispiel:  
* `docker compose up`
* `docker compose up -d`

### Compose stoppen:  
`docker compose down [OPTIONS]`

Beispiel:  
* `docker compose down`  
* `docker compose down -v` 

### Alle gestarteten Container in einer Compose:  
`docker compose ps`
